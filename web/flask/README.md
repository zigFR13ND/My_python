# Flask Web Application

В этой папке находится учебное приложение на Flask. Проект демонстрирует базовую работу с веб-фреймворком Flask, включая маршрутизацию, обработку параметров URL и возврат JSON-ответов.

## Содержимое

- **flask_app.py** – Основной файл приложения, содержащий:
  - Маршрут `/` – возвращает HTML-страницу с приветствием.
  - Маршрут `/greet/<name>` – принимает имя из URL и выводит приветствие.
  - Маршрут `/api/info` – возвращает JSON-ответ с информацией о приложении.

## Установка и запуск

1. Убедись, что установлен Python 3.x и Flask. Если Flask не установлен, выполни:
   ```bash
   pip install Flask
